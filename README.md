### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：客户、供应商、运货商、管理员。

**客户端功能：**
1. 登录
2. 注册
3. 商品一览
4. 我的订单
5. 我的订单明细
6. 安全退出


**供应商功能：**
1. 登录
2. 注册
3. 货品管理
4. 货品类别管理
5. 订单管理
6. 订单明细管理
7. 安全退出


**运货商端功能：**
1. 登录
2. 注册
3. 配送订单管理
4. 配送订单明细查询
5. 配送费管理
6. 安全退出。


**管理员端功能：**
1. 登录
2. 用户管理
3. 货品管理
4. 类别管理
5. 订单管理
6. 订单明细管理
7. 配送订单管理
8. 商品销量可视化
9. 安全退出
